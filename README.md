# -when-do-we-meet
일정 공유 및 조정 사이트 개발 용도

## 머지된 분기 요약

1. 초기 프로토타입 뼈대: 일정 맞추기 프로젝트의 기본 HTML 구조와 모바일 중심 레이아웃을 만들었습니다.
2. 달력 프로토타입 추가: 레트로 스타일의 달력 화면과 월/연도 표시, 그리드 형태의 달력 UI를 구현했습니다.
3. 테스트 구조 분리: 프로토타입과 테스트 파일을 분리하고, 파일별 역할이 나뉘도록 폴더 구조를 정리했습니다.
4. 동적 UIUX 완성: 이전/다음 달 이동, 공휴일 표시, 테마 적용이 가능한 동적 달력 UI를 구현했습니다.
